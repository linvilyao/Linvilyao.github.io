gurobi安装教程

发表于 2023-11-20 分类于 安装教程 阅读次数： 本文字数： 747 阅读时长 ≈ 1 分钟

申请

申请许可证（半年）

附件：学信网在线学籍证明、录取通知书（因为刚入学）、gurobi申请书

发送邮件：help@gurobi.cn

安装

官网：Gurobi Software - Gurobi Optimization

  1. 解压 tar -xvfz gurobi903_linux64.tar.gz
  2. 在gurobi811/linux64目录下执行 python setup.py install
  3. 许可证激活（生成gurobi.lic） /home/{自己的目录名称}/gurobi903/linux64/bin/grbgetkey 许可证
  4. ～/.bashrc配置路径 记得source ~/.barshrc
  1. 检查是否成功 gurobi.sh

  1. 另外：测试gurobi过程中出现编译器运行成功，但代码出现大量黄线。查看官方文档估计下载的gurobi本身不带类型提示功能

    解决方法：pip install gurobipy-stubs（gurobipy-stubs · PyPI）
